## Folder Descriptions

| Folder | Description |
|---|---|
| `data/` | Contains datasets in different stages |
| `data/raw/` | Original dataset files |
| `data/cleaned/` | Cleaned datasets |
| `data/processed/` | Processed datasets ready for ML |
| `notebooks/` | Jupyter notebooks for analysis and experiments |
| `model/` | Saved machine learning models |
| `spark/` | PySpark and big data scripts |
| `app/` | Dashboard or web application |
| `templates/` | HTML templates |
| `static/` | CSS, charts, and static assets |
| `visualization/` | Generated graphs and charts |
| `report/` | Final project report |
| `presentation/` | Presentation slides and demo |
| `docs/` | Documentation and diagrams |

## Technologies Used

- Python
- Pandas
- Scikit-learn
- PySpark
- Streamlit / Flask

Machine Learning and Big Data project for analyzing and predicting student grades.
